## Troubleshooting

**Bot fails to start / connection refused**
Ensure the required environment variables (e.g., API keys, host, port) are set correctly. Check the `.env` file or your shell environment and restart the service.

**Unexpected errors after pulling new changes**
Dependencies may be out of date. Run `npm install` (or the appropriate package manager command) to update them, then retry.
