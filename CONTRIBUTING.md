# Contributing

Thank you for your interest in contributing to Home Bot Dev System.

## How to Contribute

1. **Fork** the repository on GitHub.
2. **Create a branch** from `main` with a descriptive name:
   ```bash
   git checkout -b feature/your-feature-name
   ```
3. **Make your changes**, keeping commits focused and messages clear.
4. **Test your changes** to ensure the pipeline and any affected functionality still works correctly.
5. **Open a Pull Request** against the `main` branch with a clear description of what was changed and why.

## Guidelines

- Keep changes minimal and focused on a single concern per PR.
- Follow existing conventions in the codebase.
- Update documentation (README, CHANGELOG) as needed.
- Be respectful and constructive in code reviews.

## Reporting Issues

Open an issue describing the problem, steps to reproduce, and expected vs. actual behavior.
